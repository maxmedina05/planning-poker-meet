# Planning Poker — Google Meet Add-on

A lightweight Planning Poker add-on for Google Meet. Participants vote on story
points directly from the side panel during standups, without leaving the call.

## Cards

`0` `1` `2` `3` `5` `8` `13` `21` `?` `☕`

## Features

- Pick from 10 standard Fibonacci cards (0, 1, 2, 3, 5, 8, 13, 21, ?, ☕)
- Votes are hidden until the facilitator reveals them
- Live vote count updates as participants submit
- Results sorted with average and range
- Optional story title set by the facilitator
- Any participant can claim the facilitator role if the host leaves
- Start New Round resets all votes while keeping the story title
- Version number displayed in the header for quick deploy verification

## Tech Stack

| Layer | Choice |
|-------|--------|
| Hosting | GitHub Pages (static, free) |
| Deployment | GitHub Actions (injects secrets, auto-deploys on push) |
| Meet integration | Meet Add-ons SDK v1.1.0 (CDN) |
| Real-time sync | Firebase Realtime Database (CDN) |
| Language | Vanilla JS + HTML + CSS — no build step, no framework |

## Versioning

The current version is displayed next to "Planning Poker" in the side panel header.
Bump `VERSION` in `sidepanel/app.js` with each release to verify deploys at a glance.

## Secrets

`config.js` is gitignored and generated by the GitHub Actions workflow from two
repository secrets:

| Secret | Description |
|--------|-------------|
| `CLOUD_PROJECT_NUMBER` | Google Cloud project number (12-digit integer) |
| `FIREBASE_CONFIG` | Firebase web app config as a JSON object |

For local development, copy `config.example.js` to `config.js` and fill in values.
